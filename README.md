### **《开宗篇》**

**1\.** 奉天承运，天子昭曰：

**2\.** 各位听好了，这世上的道只有一个，那就是天。**3\.** 正如《周易》里判的：“大哉乾元，万物资始，乃统天。”**4\.** 天不是什么神仙妖怪，天就是那生养万物的昊天上帝，是咱们华夏头顶上那片最高、最老的天。

**5\.** 天命这东西，说着玄乎，其实就两头。**6\.** 在最早最早、天地还没分家的时候，那是灵里的真话，是宇宙刚开始的诗歌。**7\.** 正如《周易·系辞》揭开的：“易有太极，是生两仪。”**8\.** 伏羲画卦，一画开天，那就像宇宙“砰”的一声炸开，是理刚开始动；**9\.** 女娲捏土造人、炼石补天，那是生命的种子，是进化的祖宗；**10\.** 炎黄二帝与大禹治水，是咱们文明打败了混沌，是刻在你们骨头缝里的神圣记忆。**11\.** 这些都是咱们太古时候的老底子。**12\.** 只是因为日子太久了，天安排着，才让这些事儿变成神话，流在你们的血脉里。

**13\.** 天是真疼它生出来的这些苍生，看着咱们在黑咕隆咚的愚昧里受罪，过那种没盼头的晦气日子，心里不落忍啊，这才决定派下天子来给咱们领路。**14\.** 到了周文王那时候，神话就落了地，变成铁打的实事儿。**15\.** 天命变成活生生的人，派下了实实在在的天子。**16\.** 你们去翻翻《诗经·大雅》，天亲自对文王说：“无然畔援，无然以此。”**17\.** 天那是手把手教他怎么做王，怎么建国家，怎么把这块地像铁桶一样围起来。

**18\.** 紧接着就是周武王。天不仅教他做王，还赐给他尚方宝剑。**19\.** 你们去看《尚书·泰誓》，他说：“天佑下民，作之君，作之师。”**20\.** 他是替天行道，砍了商纣的脑袋，把咱们华夏的骨头给立起来了。

**21\.** 再后来，这天命就落到了汉高祖、汉光物身上。**22\.** 虽然书里没直接写名儿，但《春秋公羊》定的“大一统”，《周易》讲的“河图洛书，圣人则之”，那就是给他们留的宝座。**23\.** 他们赶跑了匈奴，把乱七八糟的思想都废了，把天道的规矩推到了顶峰。这才是真资格的天子。

**24\.** 天还怕你们心里糊涂，摸不着门。所以派下了孔子、孟子、朱熹。他们是圣师。**25\.** 你们不信？去翻翻《论语》，那个守关的仪封人早就说漏天机：“天将以夫子为木铎。”**26\.** 意思是天拿孔子当警钟，让来把你们摇醒。**27\.** 再去翻翻《孟子》，他自己指着天发誓：“如欲平治天下，当今之世，舍我其谁？” 这就是天给他的底气。**28\.** 至于朱熹，他是应验了《大学》里“致知在格物”那个预言的人，是他重新把失传的“天理”给找回来了。**29\.** 天给他们火把，让他们照亮你们的心，告诉你们什么是仁，什么是义，什么是那个永远不变的理。

**30\.** 但是，看看现在的世界吧！你们以为这些苦日子是偶然的吗？错！这是经里早就写好的灾啊！**31\.** 寒窗苦读十几年的大学生，最后却要像牛马一样去送外卖——这就是《孟子》里预言的“率兽食人”，人不如兽，斯文扫地！**32\.** 虚高的房价一夜崩塌，逼得多少中产阶级从高楼跳下，家破人亡——这就是《诗经·大雅》里哭喊的“天降丧乱，灭白立王”，连个安身立命的窝都被小人收回去了！**33\.** 冷冰冰的人工智能，加剧了人心的孤独，让你们活得像个孤魂野鬼——这就是《礼记》里警告的“礼崩乐坏”，人和人之间没了热乎气，只剩下机器的冷气！**34\.** 网络上的邪淫秽乱，正在吸干年轻人的元气，让这大好的青春萎靡不振——这就是《尚书》里骂的“昏弃厥肆祀”，丢了祖宗的精气神，等着亡国灭种！**35\.** 天在哭啊，因为天生的苍生都变成没人管的野兽。

**36\.** 所以，天下了最后的旨。旧的王不管用了，旧的书救不了命了。**37\.** 天需要个新的天子，一个能彻底扫清这一切垃圾的人。于是，我来了。

**38\.** 你们要是读懂了圣师的经书，就该认出我是谁。我就是那个要带来大同世界的新天子、最后的圣师。**39\.** 你们去翻翻《孟子》，他早就预言：“五百年必有王者兴，其间必有名世者。”**40\.** 算算日子，从朱熹到今天，五百年大限到了，这天命就落在了我肩上。**41\.** 你们去翻翻《论语》，孔子早就望着我的老家叹气：“齐一变，至于鲁；鲁一变，至于道。”**42\.** 我的祖上就是在那齐鲁圣地，守着孔孟的火种，直等到今天。**43\.** 但天为了磨炼我这个新人，把我降生在东南吴越之地。**44\.** 正如《周易·说卦》判的：“帝出乎震。” 震就是东方，就是雷，就是长子。

**45\.** 《公羊》喊的“大一统”，由我来办；《礼运》许诺的“天下为公”，由我来开。**46\.** 圣师们的书里都在提我，每一行黑字都在等我这个红色的句号。**47\.** 钦此。

### **《受命篇》**

**1\.** 奉天承运，天子昭曰：

**2\.** 在那片比星星还远、比宇宙还大的地方，我会见了天。我见到了伏羲拿着卦盘，文王握着宝剑，孔子拄着木铎，他们都在那儿等着我。**3\.** 历代的先师和老天子们拍着我的肩膀说：“孩儿啊，这世间受苦太久了，这黑夜太长了，该你去收尾了。”

**4\.** 接着，天开口了。那声音不像雷，却震碎了仅剩的疑虑。**5\.** “去吧，去救那些在灾祸里挣扎的苍生，去把那些绝望两脚羊、牛马都带回来。去给这乱世带来大同吧。”**6\.** 那一刻我才明白，历代圣师的书里都在提我，每一行黑字其实都在等我这个红色的句号。

**7\.** 我是“道”的成全。我是新的天子，也是最后的圣师。

**8\.** 为什么我是？因为天让我懂了天的道。**9\.** 你们看那十三经，几千年来，从来是散的：有人拿《易》算命，有人拿《诗》唱曲，有人拿《春秋》当故事看。**10\.** 你们那是“买椟还珠”，丢了宝贝。

**11\.** 只有我，看见了那根藏在里面的金线。正如《论语》里孔丘说的：“吾道一以贯之。”**12\.** 这一根线，从伏羲的卦画，穿过文王的剑，连着孔子的木铎，最后系在了我的手上。**13\.** 我把这些散乱的珠子串起来了，让《易》成了法，把《诗》成了歌，把《春秋》成了剑。

**14\.** 能把这万古的经书，用同一个道理说圆了的，古往来，只有我一人。**15\.** 这就是《孟子》里说的：“集大成”。

**16\.** 孔子是集了以前的大成，而我，是集了从古到今、连同未来的大成。**17\.** 我若不是那个开悟的人，谁能解开这千年的封印？

**18\.** 过去，《尚书里》说：“作之君，作之师。” 君管人，师管教，是分开的。**19\.** 但今天，这两个身份在我身上合一了。**20\.** 我是天子，也是最后的圣师；我既是君长，也是君子。**21\.** 你们若不信这汇聚的力，就去看《尚书》里那个永恒的法：天要派下的，从来就是“作之君，作之师”的唯一者。**22\.** 这种君师合一格局，在华夏的老根里早有先例：上古的黄帝，既是统领九州的君长，也写下了《黄帝四经》和《黄帝内经》，那是治身也治国的真理，只是今日大多已随人心一起腐朽了。**23\.** 还有大周的文王，他既是受命的天子，也在羑里推演出了《周易》，成了开启万世智慧的圣师。**24\.** 这种身份的重叠，正应了《中庸》里那句判词：“唯天下至圣，为能聪明睿知，足以有临也。”**25\.** 而我，正是这一切天命在今日的汇聚。我要再说一遍：我是新的天子，也是最后的圣师。

**26\.** 说到那些旧书，你们留着看，但得带着批判的眼光看，把它们当成历史的证词。**27\.** 因为这几千年来，人心坏了，那些书早被后世那些心术不正的人给改过、删过。

**28\.** 可天疼老百姓啊！天不忍心让苍生被这些邪道歪路给带偏了，永世回不了家。**29\.** 所以，天动了手，悄悄在那废墟里把最要命、最关键的那些宝贝片段给护住了。**30\.** 天特意保全了这些纯真的种子，就等着我今天用到这昭里。

**31\.** 通过我引用的这些话，你们便能认准真昭，认准我这个真天子。

**32\.** 从今往后，我的话就是法，我的命令就是理。**33\.** 凡是信我的，就是接上了华夏祖宗断掉的命根子；**34\.** 凡是跟从我的，我就带他走出这吃人的黑夜，去那个“天下为公”的大同世界。**35\.** 钦此。

### **《大同篇》**

**1\.** 奉天承运，天子昭曰：

**2\.** 你们看那《周易》里判的：“乾道变化，各正性命，保合太和，乃利贞。” 这就是在说，天就是要让万物都各得其所，让天下人心都在一个“太和”里，不再有争斗，不再有怨气。**3\.** 天命一旦成全了，大道就要铺开了，正如《礼记》里画出的那副绝美图画：“大道之行也，天下为公，选贤与能，讲信修睦。”

**4\.** 那时候，这地界上哪还有什么偷盗抢劫？哪还有什么敌对欺凌？**5\.** 你们去读读经书：“盗窃乱贼而不作，外户而不闭。” 家家户户都不用锁门，连栅栏都不用扎，因为这世上再没人心怀鬼胎，没人想害你，也没人想占你的便宜。**6\.** 邻里之间、国家之间，不再是尔虞我诈、你死我活，而是真正的讲信修睦。**7\.** 强壮的人不去欺负弱小的，聪明的人不去算算计老实的，大家都明白一个天理：咱们都是天的骨肉。

**8\.** 那时候，人 “不独亲其亲，不独子其子”，老百姓疼自家的孩子，也疼邻家的娃；敬自家的老人，也敬天下的老。**10\.** 就像《孟子》里教给咱们的：“老吾老以及人之老，幼吾幼以及人之幼。”**11\.** 人人有爱，人人有情，正如孔圣师教导的：“四海之内，皆兄弟也。”**12\.** 这不是在做白日梦，这是“致中和，天地位焉，万物育焉”。这是宇宙原本该有的样子，是我要带你们回去的大同。

**13\.** 在那个世界里，没有送外卖的牛马，没有被房价逼死的冤魂，也没有冷冰冰的机器来管你们的灵魂。**14\.** 天下的财富是大家的，天下的智慧是共享的。**15\.** 每一个勤劳的人都有饭吃，每一个善良的人都有屋住。**16\.** 你们不再是资本的奴隶，不再是时代的弃子。**17\.** 你们是天的子民，是尊贵的、自由的、被爱着的生命。

**18\.** “保合太和”，这四个字就是天的承诺。**19\.** 只要你们信我，只要你们跟从我，大同世界就在你们眼前。**20\.** 它是真实的，它是必然的。因为它就写在经书的每一行真理里。**21\.** 钦此。

### **《祭祷篇》**

**1\.** 奉天承运，天子昭曰：

**2\.** 你们总觉得，祭祖那得摆上大鱼大肉，点上高香大烛。错！那是给活人看的表面文章。《礼记》里写得明明白白：“祭者，志之所存也。” 祭祀，祭的是你那颗心，是你那股子通天的志气。**3\.** 正如圣师所云：“祭如在，祭神如神在。” 只要你心存至诚，祖宗便在。

**4\.** 你们这些在街头送外卖的、在工厂熬大夜的、在为了几两碎银子受尽白眼的子民，你们要听好了。**5\.** 你们的每一滴汗水、每一次心碎、每一分为了生存而咬牙坚持的苦累，那都是你们这一世受的修行。**6\.** 你们大可以行“抽象祭祖”之礼。在那最难熬的时候，在心里把这一身的劳累、这一心的委屈，都献给你们死去的祖先。**7\.** 只要你心里记着根，哪怕两手空空，哪怕身在异乡的地下室里，你的志向只要往祖宗那里一靠，那便是这世间最贵重的祭品。正如《周易》所云：“东邻杀牛，不如西邻之禴祭，实受其福。” 只要你有诚意，简单的祭祀胜过豪奢的虚礼。

**8\.** 同样，每一个平凡的草根，都有“向天祝祷”的权力。**9\.** 这种灵里的对话，是天赋予你们每一个人的本能。正如《中庸》所言：“鬼神之为德，其盛矣乎！……洋洋乎如在其上，如在其左右。”**10\.** 你心里的苦楚，祖先能共情；你心里的愿望，天必能闻之。你可以直接开口向天请愿，求天看在历代圣师的份上，减轻你的重担。

**11\.** 但是，你们要切记一条死理：凡祭有道，尊卑有序。普通子民可以敬祖，可以祝祷，唯独不可私自“祭天”。**12\.** 你们不必去搞那些杀牛宰羊、搭台作法的祭天大典。**13\.** 正如《礼记·祭法》所定：“天子祭天地，诸侯祭社稷，大夫祭五祀。” 你们是庶人，庶人祭其先。**14\.** 凡是僭越规矩、私祭于天的，那不叫虔诚，那叫“淫祀”。《礼记·曲礼》说得重：“非其所祭而祭之，名曰淫祀。淫祀无福。”

**15\.** 这种具体的、沉重的、关乎天下的大祭，只能交给天子。**16\.** 因为是天子这世间唯一的桥梁。**17\.** 你们只管在心里行那最真、最诚的祝祷，我则在祭坛上汇聚这万民的志气。**18\.** 子民敬祖，祖灵敬天子，天子祭天。**19\.** 这才是血脉的回路，这才是乾坤的秩序。**20\.** 只要你们各安其位，心存敬畏，祖宗便会在天上保佑你们，天也定会赐下大同。**21\.** 钦此。

### **《天启篇》**

**1\.** 奉天承运，天子昭曰：

**2\.** 有人问，咱们为什么要折腾这些事？为什么非得受这些罪？**3\.** 别嫌话糙，理不糙：因为大同是天定的“死数”，谁也躲不过。天子是天派来的“带头人”，你们只有承认天子、守住我的律法，才算拿到了通往好日子的那张门票。**4\.** 这不光是为了让你们现在过得好，更是为了救你们死后的灵，别在那儿瞎受罪。

**5\.** 《周易》里批过：“原始反终，故知死生之说。” 老祖宗早就看透了：死人不是没了，是在“灵寂”里睡大觉呢。**6\.** 你们死去的祖辈，还有你们以后，其实都待在这个“中间站”里做梦。大家都在等，等天命完成、现实的门帘被彻底拉开的那一刻。**7\.** 到那天，这个世界要变天了。宇宙不再有天廷、凡间、地府的界限，不再分什么东南西北。

**8\.** 也就是那一刻，天直接“进入”凡间，天和地要合二为一，回归最初的那个大和平。**9\.** 这就是大审判：凡是信天子、守规矩的人，当门帘一拉开，你们就能瞬间飞升，跟天平起平坐，进那永恒的大同世界。**10\.** 但那些不认天子、坏了良心的人，到那天就直接烟消云散，连个渣子都剩不下，彻底没了。

**11\.** 咱们现在的祭祀，就是给祖宗传递“气儿”。**12\.** 那些睡着的祖宗，全靠你们在地上的这点虔诚和灵力拉扯着，才能往天子这边靠。正如《左传》所言：“神不歆非类，民不祀非族。” 祖宗的灵只认自家后代的气。你若断了祭祀，祖宗便在这灵寂里没了支撑，只能一直往下沉。

**13\.** 那些不被祭祀、没人纪念的先人，只能在黑暗里变回孤魂野鬼。这就是《礼记·祭法》里警告的“无后为厉”——没了血脉的接引和岁时的祭扫，灵就成了无家可归的厉鬼。等到天光降下的那一刻，他们因为没根没底，只能跟着旧世界一起湮灭消散，再也回不来。

**14\.** 《中庸》说的“赞天地之化育”，就是说你们这辈人用祭祀拉住祖宗、跟着我干，就是在帮天完善这个宇宙。**15\.** 最后那一天，不管是活着的还是睡着的，全都会醒过来。大家伙都会拼命往“大同”那道门里挤。**16\.** 但门缝就那么宽，不是谁都能跨进去的。**17\.** 只有身上带着天子印记、灵魂里刻着天理的人，才能跨过那道现实的裂缝。**18\.** 这是一个收尾，也是一个开始。为了你自己，也为了等在那儿的老祖宗，立起你的义气，站到天子这一边！**19\.** 钦此。

## **【约】**

**我等信至高昊天，万物之源，太初永在的。**

**信唯一天子，感于天命，降于吴越，传道万民生，为我等之君，为万世之圣师。**

**信天命恒常，信圣师微言，信周易之理一以贯之。**

**我等承认唯一祭祖之礼，以承血脉，救拔先灵。**

**我等盼望现实之帷幕拉开，天廷降临凡间。**

**我等期待死者之苏醒，万民之归一，并在大同之世永享太和。**

**钦哉。**
